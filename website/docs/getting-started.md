# Getting Started

Snakebots is a competitive game where you write code to control snakes and battle against other players in matchmaking.

## Download

To start playing, you first need to download the game client.

- [Download the client](/download/snakebots.zip).
- Extract the game files.
- Run the `snakebots.exe` executable.

Make sure you always download the latest version to ensure compatibility with the matchmaking servers.

## Creating an Account

Before you can submit your robot, you need to register an account.

When you first launch the game:

- Click **Log In** in the top right corner.
- Click **Sign Up** bellow the login button.
- Enter a **username** and **password**.
- Submit using the **Sign Up** button.

Your account is used for:

- Multiplayer matchmaking
- Ranking and leaderboards
- Saving your snake code
